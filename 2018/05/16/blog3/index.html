CSS3系列之一：border-radius

发表于 2018-05-16 | 分类于 CSS3学习笔记系列 |
字数统计: 187 | 阅读时长 ≈ 1

border-radius是CSS3的属性之一，作用是为元素添加圆角的边框。
border-radius只有在以下版本的浏览器中使用Firefox4.0+、Safari5.0+、Google Chrome 10.0+、Opera 10.5+、IE9+ 。对于老板浏览器，需要根据不同的浏览器内核添加不同的前缀，比说 Mozilla 内核需要加上“-moz”，Webkit 内核需要加上“-webkit”等，为了最大程度的兼容浏览器，可设置如下：
-webkit-border-radius: 10px 20px 30px;
-moz-border-radius: 10px 20px 30px;
border-radius: 10px 20px 30px;
但是很久以前浏览器就已经把私有前缀去掉了，一般新版本的浏览器都会支持border-radius，现在完全没必要进行兼容处理了。
